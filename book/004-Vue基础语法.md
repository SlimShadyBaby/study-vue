# Vue基础语法

## 环境安装

1. 安装node.js和npm
2. 安装vue cli: npm install -g @vue/cli
3. 创建vue项目: vue create project_name
4. 运行vue项目: [npm run serve](../images/run.png)

## 开发工具

vscode,插件：

1. HTML CSS Support
2. Vetur

## 项目结构

## 页面结构

1. template: html定义
2. script: vue逻辑代码区域
3. style: 样式定义, css

## 模板语法

### 插值

1. {{ value }}
2. v-html指令
3.Attribute

### 指令

1. 以v-开头的attribute,如v-if, v-bind, v-on等
2. v-bind缩写为":",如：href
3. v-on缩写为"@",如：@click
